The Smart Maintenance Management System is a full-stack web application designed to digitize and automate the process of handling maintenance requests in institutions such as universities, hostels, apartments, or corporate facilities.

The system provides a centralized platform where users can raise maintenance requests, technicians can manage assigned tasks, and administrators can monitor, assign, and analyze maintenance operations efficiently.
