Smart Maintenance Management System

Smart Maintenance System is a role-based web application built to manage maintenance issues in a campus or organization in a simple and efficient way.

The system allows students to raise maintenance requests, administrators to manage and assign those requests, and technicians to complete the work while updating the status. The goal of this project is to reduce manual communication and make maintenance tracking clear and organized.

About This Demo Version

This website shows a frontend demo version of the Smart Maintenance System and is hosted using GitHub Pages.

GitHub Pages supports only static websites, so the PHP backend and MySQL database cannot run here. Because of this, the original PHP pages have been converted into static HTML demo pages so that the interface and workflow can still be viewed and understood.
