Smart Maintenance System

Project Overview

The Smart Maintenance Management System is a role-based maintenance tracking platform designed to streamline the process of raising, assigning, and resolving maintenance requests.

This live demo showcases the frontend UI hosted on GitHub Pages. The complete backend functionality is implemented using PHP and MySQL and runs locally via XAMPP.
